# Zentris Global - Home estática

## Cómo ejecutar
- Abre `index.html` directamente en tu navegador.
- O usa una extensión tipo Live Server para servir la carpeta.

## Checklist QA responsive
- 360px: Header y CTAs no se rompen, botones con alto >= 44px.
- 390px: Hero y tarjetas con buena jerarquía visual.
- 430px: Pills y botones con saltos controlados.
- 768px: Layout en dos columnas en hero y comparación.
- 1024px: Logo a 80px de alto, grillas completas.

## Checklist de contraste
- Texto principal (#0B1220) sobre blanco y gris (#FFFFFF / #F4F4F4).
- Texto secundario (#4B5563) sobre blanco y gris.
- Botón primario (#004AAD) con texto blanco.
- Botón secundario (borde #004AAD) con texto azul sobre fondo blanco.
- Callout con borde verde (#A3E635) y texto #0B1220.
